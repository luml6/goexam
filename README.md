## 云笔记后台
- 针对用户需求开发的一款自主研发的最新云笔记。包含用户所有操作的后台代码。

>|目录 | 描述|

>| ├── comm          |常用操作

>| │   └── myredis   |redis操作
>| ├── conf          |配置文件
>| ├── controllers   |beego controller
>| ├── docs          |文档目录
>| ├── model         |数据层
>| │   └── tests     |数据层单元测试
>| ├── operatelog    |用户行为日志存储
>| ├── routers       |beego路由
>| ├── services      |业务服务层
>| │   └── tests     |服务层单元测试
>| └── tests         |API单元测试
>| └── utils         |工具类目录
